## 1. 流程控制 + 异常捕获

#### 1.1 流程控制

在Python中，流程控制主要包括 **条件语句** 和 **循环语句**。

- **条件语句（if-elif-else）**

  在C和C++中的写法为`if`、`else if`和`else`。Python中的写法略有不同，使用`if`、`elif`和`else`。

  **示例：**

  python

  ```
  x = 10
  if x > 0:
      print("x is positive")
  elif x == 0:
      print("x is zero")
  else:
      print("x is negative")
  ```

- **循环语句（for/while）**

  **for循环**：

  - 在C和C++中，`for`循环的语法为 `for (初始化; 条件; 更新)`。
  - Python中的`for`循环更简洁，常用于遍历序列（如列表、元组、字符串等）。

  **示例：**

  python

  ```
  for i in range(5):  # 类似于C/C++中的 for (int i = 0; i < 5; i++)
      print(i)
  ```

  **while循环**：

  - Python的`while`循环与C和C++的用法相同，基于条件进行循环。

  **示例：**

  python

  ```
  i = 0
  while i < 5:
      print(i)
      i += 1
  ```

#### 1.2 异常捕获（try-except）

异常处理在编程中非常重要，用于处理运行时可能出现的错误。

- **基本语法**：

  python

  ```
  try:
      # 可能引发异常的代码
      result = 10 / 0
  except ZeroDivisionError:
      # 处理特定类型的异常
      print("除数不能为零")
  except Exception as e:
      # 处理其他类型的异常
      print(f"发生错误: {e}")
  else:
      # 没有发生异常时执行的代码
      print("操作成功")
  finally:
      # 无论是否发生异常都会执行的代码
      print("执行完毕")
  ```

## 2. 推导式 + 函数式编程

#### 2.1 列表推导式

列表推导式是一种简洁的方式来创建列表。

- **基本语法**：

  python

  ```
  new_list = [expression for item in iterable if condition]
  ```

  **示例：**

  python

  ```
  squares = [x**2 for x in range(10)]  # 创建一个包含0到9的平方的列表
  even_squares = [x**2 for x in range(10) if x % 2 == 0]  # 只包含偶数的平方
  ```

#### 2.2 匿名函数（lambda）

匿名函数即`lambda`函数，用于定义简单的、一次性使用的函数。

- **基本语法**：

  python

  ```
  lambda arguments: expression
  ```

  **示例：**

  python

  ```
  add = lambda x, y: x + y
  print(add(5, 3))  # 输出 8
  ```

#### 2.3 高阶函数（map, filter）

- **map函数**：对序列中的每个元素应用一个函数。

  **示例：**

  python

  ```
  numbers = [1, 2, 3, 4]
  squared = list(map(lambda x: x**2, numbers))  # [1, 4, 9, 16]
  ```

- **filter函数**：根据条件过滤序列中的元素。

  **示例：**

  python

  ```
  numbers = [1, 2, 3, 4, 5]
  even_numbers = list(filter(lambda x: x % 2 == 0, numbers))  # [2, 4]
  ```

### 总结

- **流程控制**：Python的`if-elif-else`和`for/while`循环语法简洁，易于理解和使用。
- **异常捕获**：通过`try-except`结构可以有效处理程序运行时的异常。
- **推导式**：列表推导式提供了一种简洁的方式来创建列表。
- **函数式编程**：`lambda`函数和`map`、`filter`等高阶函数使得代码更加简洁和高效。
