## 一、矩阵转置（Transpose of a Matrix）

### 1. **定义**

- **转置**：把矩阵A的行换成相应的列，得到的新矩阵称为A的转置矩阵，记作A^T或A'。
- 维度变化：m×n矩阵的转置是n×m矩阵。
  - 例如：A*=[*a*11*a*21*a*12*a*22] （2×2矩阵）的转置是 *A*^T*=[*a*11*a*12*a*21*a*22*]

### 2. **基本性质**

| 性质          | 公式                         | 说明                 |
| :------------ | :--------------------------- | :------------------- |
| 1. 转置的转置 | (*A^T*)^*T*=*A*              | 转置两次等于原矩阵   |
| 2. 和的转置   | (*A*±*B*)*T*=*A*^*T*±*B*^*T* | 逐元素操作           |
| 3. 乘积的转置 | (*A* *B*)^*T*=*B*^*T* A^*T*  | **重要**：顺序反转   |
| 4. 数乘的转置 | (*k* *A*)^*T*=*k* *A*^*T*    | 标量k可提出          |
| 5. 行列式不变 | det(*A*^*T*)=det(*A*)        | 转置不改变行列式的值 |

### 3. **特殊转置矩阵**

| 类型       | 定义                      | 例子                               |
| :--------- | :------------------------ | :--------------------------------- |
| 对称矩阵   | *A*^*T*=*A*               | *A*=[1 2 2 3]                      |
| 正交矩阵   | *A**A*^*T*=*A*^*T**A*=*E* | *A*=[cos*θ* sin*θ* −sin*θ* cos*θ*] |
| 斜对称矩阵 | *A*^*T*=−*A*              | *A*=[0 2 −2 0]                     |

### 4. **为什么重要？**

- 机器学习应用：
  - 点积计算：v**⋅**w**=**v*^*T**w**
  - 矩阵乘法中维度匹配：确保 *A*^*T* *B* 可计算
  - 优化算法：如梯度下降中计算梯度（ ∇*f*=**X**^T(**Xw**−**y**) ）

 **tips**：矩阵乘积的转置要**反转顺序**， (*A* *B*)^*T*=*B*^*T* *A*^*T* .

## 二、矩阵的逆（Inverse of a Matrix）

### 1. **定义**

- **逆矩阵**：若存在矩阵B使得 A**B*=*B**A*=*E（E为单位矩阵），则B称为A的逆矩阵，记为 A^−1 。
- **可逆条件**：矩阵A可逆的**充要条件**是其行列式∣*A*∣≠ 0 （即A是**非奇异矩阵**）。
- **注意**：仅**方阵**（行数=列数）有逆矩阵概念。

### 2. **等价条件（n阶方阵A可逆）**

| 条件                               | 说明               |
| :--------------------------------- | :----------------- |
| 1. $                               | A                  |
| 2. rank(*A*)=*n*                   | 满秩（秩等于阶数） |
| 3. A的行向量组线性无关             | 行向量不能互相表示 |
| 4. A的列向量组线性无关             | 列向量不能互相表示 |
| 5. 齐次方程 *A* *x*=0 只有零解     | 无非零解           |
| 6. *A* *x*=*b* 对任意 *b* 有唯一解 | 方程有唯一解       |

### 3. **逆矩阵的性质**

| 性质        | 公式                            | 说明               |
| :---------- | :------------------------------ | :----------------- |
| 1. 逆的逆   | (*A*^(−1)^(−1)=*A*              | 逆的逆是原矩阵     |
| 2. 数乘的逆 | (*k* *A*)^(−1)=*k*^(-1)*A*^(−1) | *k*≠0              |
| 3. 乘积的逆 | (*A* *B*)^(−1)=*B*^(−1)*A*^(−1) | **重要**：顺序反转 |
| 4. 转置的逆 | (*A*^*T*)^(−1)=(*A*^(−1)^*T*    | 转置和逆可交换     |

## 三、行列式（Determinant）

### 1. **定义**

- 行列式是n阶方阵A的一个**标量值**，记作det(*A*) 或∣*A*∣ 。
- **几何意义**：在n维空间中，行列式表示线性变换对"体积"的影响（如2D中表示有向面积）。

### 2. **基本性质**

| 性质        | 公式                                    | 说明                                                         |
| :---------- | :-------------------------------------- | :----------------------------------------------------------- |
| 1. 转置不变 | det(*A*^*T*)=det(*A*)                   | 转置不改变行列式值                                           |
| 2. 行列交换 | 互换两行（列），行列式变号              | 例如： det[*a* *c* *b* *d*]=−det[*c* *a* *d* *b*]            |
| 3. 数乘行   | 某行乘以k，行列式乘以k                  | det[*ka* *c* *kb* *d*]=*k* det[*a* *c* *b* *d*]              |
| 4. 成比例行 | 两行成比例，行列式=0                    | 例如： det[*a* 2*a* *b* 2*b*]=0                              |
| 5. 行列相加 | 一行是两数之和，行列式可拆              | det[*a*1+*a*2 *c* *b*1+*b*2 *d*]=det[*a*1 *c* *b*1 *d*]+det[*a*2 *c* *b*2 *d*] |
| 6. 行列变换 | 行（列）乘k加到另一行（列），行列式不变 | 用于计算行列式                                               |

### 3. **行列式的计算（2×2矩阵）**

det[*a* *c* *b* *d*]=*a* * *d* − *b* * *c*

- **可逆条件**：*a* * *d* − *b* * *c* ≠ 0

### 4. **为什么重要？**

- 机器学习应用：

  - **判断矩阵可逆**：∣*A*∣≠ 0 是矩阵可逆的充要条件

  - **特征值计算**：特征方程 det(*A*−*λ* *I*)=0

  - 雅可比行列式

    ：在坐标变换中（如极坐标到直角坐标的转换）用于面积/体积缩放

    - 例如：极坐标变换中， d* *x* *d* *y*=∣det(*J*)∣*d* *u* *d* *v* ，其中 �*J* 是雅可比矩阵

## 四、关键总结与易错点

| 概念         | 核心要点                                                    | 常见错误                           |
| :----------- | :---------------------------------------------------------- | :--------------------------------- |
| **矩阵转置** | (*A* *B*)^*T*=*B*^*T* *A*^*T* （顺序反转）                  | 混淆 (*A* *B*)^*T*=*A*^*T* *B*^*T* |
| **矩阵的逆** | 仅方阵有逆；(*A* *B*)^(−1)=*B*^(−1) (*A*)^(−1) （顺序反转） | 对非方阵求逆                       |
| **行列式**   | $                                                           | A                                  |

------

## 五、今日小结

- **矩阵转置**：行变列，(*A* *B*)^*T*=*B*^*T* *A*^*T* 。
- **矩阵的逆**：仅方阵有逆， (*A* *B*)^(−1)=*B*^(−1)*A*^(−1) 。
- **行列式**：∣*A*∣≠ 0 是矩阵可逆的充要条件，几何意义是体积缩放。
